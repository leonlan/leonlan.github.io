Leon Lan

Hey, I'm Leon! I am a third-year PhD candidate at the Department of Mathematics of VU Amsterdam. I work on models and algorithms for real-world machine scheduling and vehicle routing problems.

Software

PyVRP
An open-source, state-of-the-art vehicle routing problem solver in Python.
VRPLIB
Python library to read and write vehicle routing problem instances.
PyJobShop
Constraint programming for real-world job shop scheduling problems.
ALNS
Python implementation of the adaptive large neighborhood search (ALNS) metaheuristic.
