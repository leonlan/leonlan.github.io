Introduction

A research tech stack as the set of technologies that a researcher uses to do research. In my personal research this includes tools for reading, writing, programming and managing. The list below is an accumulation of tools that I have found to be productive in my research work.

Research Tech Stack

Reading

Zotero
Open-source bibliography management software. The bread-and-butter of my research stack.
Google Scholar
Besides being a helpful search engine to look for papers, it also provides a way to stay up-to-date with the latest research publications.
Citation Gecko
A search tool to find the most relevant papers to my current research.

Writing

LaTeX
Markup language to write mathematical (and non-mathematical!) documents.
Overleaf
An online LaTeX editor for collaborative writing. Useful in collaborative research projects.
iA Writer
Markdown text editor for a visually pleasant writing experience.
Org mode
Markup language for creating rich plain text notes. More powerful than Markdown.
Zettelkasten (org-roam)
Personal knowledge management system for note-taking. I use org-roam in Emacs, but Obsidian is a good markdown-based alternative.

Programming

Python
My first programming language and also the one I'm most fluent in. I use it for rapid prototyping.
Julia
As easy as Python, but as fast as C++. I use Julia when I need better computational performance.
Git
Version control tool to 1) back-up everything I do and 2) to collaborate with others on software. I mainly use GitHub.
Emacs
My favorite text-editor. I use the Doom framework.

Managing

Todoist
Task management software. I use this on a daily basis for short-term projects and general to-do management.
Airtable
Project management software. I use this mainly to get a bird's eye view on my long-term projects and to make Gantt charts. Free 2-year pro trial available for students here.
Slack
Easy-to-use communication platform with support for threads to keep discussion channels free from clutter.

Other

Inoreader
RSS feed reader to keep track of the latest blog posts.
Twitter
A fun and informal way to stay up to date with academia.
